🚀 QUICK START

1. Clone Repository:
   git clone https://github.com/seanebones-lang/AGENTICteam.git
   cd AGENTICteam

2. Add API Keys:
   cd backend
   cp .env.example .env
   # Edit .env and add your LLM API key

3. Start Services:
   docker-compose up -d

4. Access Platform:
   Backend API: http://localhost:8000
   API Docs: http://localhost:8000/docs
   Frontend: http://localhost:3000
